Most of these issues are still under developing, sorry for inconvenience

  • Equal button:

    When passing the equal button to get computation, the calculator will be in clear state. You cannot pass new number to start restart the calculator, you need to click 'C' button to clear the current state. However, you can continue to use the previous result for further computation.

  • Plus/Minus Button

    When pressing Plus/Minus button, you will be brought back to the clear state of the calculator. It will ask you to pass a number in order to get its minus value.

  • Percentage Button

    Same issue with Plus/Minus button
